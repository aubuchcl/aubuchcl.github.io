What is SQL injection
SQL injection is when someone maliciously enters information into a field that will potentially cause submission of said form to delete, drop, or dump parts or all of the database. This will generally happen when an application doesnt properly validate the data being submitted by a user. It is a problem because the attacks can reveal information that should otherwise be secured in the database such as personal social security numbers, bank accounts, and so on. The only way to combat something like this is to have validation measures. To validate the data you must limit the input that can be submitted by users.
